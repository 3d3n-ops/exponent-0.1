# Quick Start

## 1. Install and Authenticate

```bash
pip install exponent-ml
exponent login
```

## 2. Create Your First Model

```bash
exponent init quick "classify emails as spam or not spam" --dataset emails.csv
```

## 3. Train Your Model

```bash
exponent train
```

## 4. Deploy to GitHub

```bash
exponent deploy
```

That's it! Your ML model is now trained and deployed.
